# Brand assets

These icon and logo images are a mirror of the official Home Assistant brand
assets for the `wattpilot` integration, published in the
[home-assistant/brands](https://github.com/home-assistant/brands/tree/master/custom_integrations/wattpilot)
repository. Home Assistant serves them from its brands CDN based on the
integration domain; they are kept here for reference and offline use.

| File          | Dimensions | Purpose            |
|---------------|------------|--------------------|
| `icon.png`    | 256×256    | Square icon        |
| `icon@2x.png` | 512×512    | Square icon (hDPI) |
| `logo.png`    | 924×256    | Wide logo          |
| `logo@2x.png` | 1847×512   | Wide logo (hDPI)   |

To change the branding, submit the updated images to the upstream
`home-assistant/brands` repository (that is the source of truth) and refresh the
copies here.
